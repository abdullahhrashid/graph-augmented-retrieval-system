This project utilizes the **HotpotQA** dataset, a large scale dataset for diverse, explainable multi-hop question answering. I would like to thank the authors for making this resource available to the research community.

* **Project Website:** [hotpotqa.github.io](https://hotpotqa.github.io/)
* **Original Paper:** [HotpotQA: A Dataset for Diverse, Explainable Multi-hop Question Answering](https://arxiv.org/abs/1809.09600)
